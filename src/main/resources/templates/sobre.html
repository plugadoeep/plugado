Objetivo

Estabelecer relações entre alunos para grupos de estudos juntamente a organização de ambientes físicos para o desenvolvimento das atividades desejadas.

Motivação

Atualmente na Escola de Engenharia de Piracicaba existe uma carência entre os alunos para organização e agendamento para grupos de estudo juntamente com a utilização de salas de aula ou biblioteca. Nosso produto propõe um meio eficaz para estabelecer uma organização entre os elementos citados.
